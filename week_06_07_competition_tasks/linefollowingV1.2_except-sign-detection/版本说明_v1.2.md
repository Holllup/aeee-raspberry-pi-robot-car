# 巡线 V1.2 版本说明

## 代码文件

- 当前归档代码: `line_following_minimal_v1_2.py`
- 当前树莓派运行路径: `/home/jacob/line_following_minimal.py`

## 本版本主要功能

1. 保留原有基础巡线功能。
2. 保留右侧紫色块检测和原有舵机动作逻辑。
3. 新增画面中部横向检测带的粉紫色/紫色检测。
4. 当检测带碰到目标颜色时:
   - 小车立即停止
   - 舵机转到 `120` 度
   - 等待 `5` 秒
   - 舵机回到 `180` 度
   - 小车恢复巡线
5. 新增调试视频录制功能，可保存带检测框、检测带和状态字样的处理后视频。
6. 优化直角弯处理:
   - 检测到锐角/直角进入趋势时提前降速
   - 增大左右轮差，提高转弯力度
   - 丢线后的角点恢复阶段使用更强的转向参数

## 本版本针对现场做过的关键调整

### 1. 颜色检测

- 中部横向检测带位置: `CENTER_PURPLE_SCANLINE_Y_RATIO = 0.62`
- 检测带半高度: `CENTER_PURPLE_BAND_HALF_HEIGHT = 8`
- HSV 阈值:
  - `H: 118 - 179`
  - `S >= 20`
  - `V >= 100`
- 目标是适应现场偏粉紫、轻微虚焦、亮度变化的实拍画面。

### 2. 紫色触发停车动作

- 停车后舵机角度: `CENTER_PURPLE_STOP_ANGLE = 120`
- 停车等待时间: `CENTER_PURPLE_STOP_WAIT_SECONDS = 5.0`
- 冷却时间: `CENTER_PURPLE_TRIGGER_COOLDOWN_SECONDS = 6.0`

### 3. 直角弯修复

- 普通巡线基础速度: `BASE_SPEED = 24`
- 锐角进入时参数:
  - `SHARP_ENTRY_BASE_SPEED = 16`
  - `SHARP_ENTRY_STEERING_GAIN = 60.0`
- 锐角丢线恢复时参数:
  - `SHARP_CORNER_PIVOT_SPEED = 40`
  - `SHARP_CORNER_PIVOT_TIMEOUT_SECONDS = 0.95`
  - `SHARP_RECOVERY_TURN_SPEED = 34`
  - `SHARP_RECOVERY_FORWARD_SPEED = 0`
