## 2. 接线说明

当前 LCD1602 使用 4 位并口模式。

### 2.1 控制线与数据线

接线关系如下：

| LCD | wiringPi | BCM GPIO | 物理引脚 |
| --- | --- | --- | --- |
| RS | 3 | GPIO22 | Pin 15 |
| E  | 0 | GPIO17 | Pin 11 |
| D4 | 6 | GPIO25 | Pin 22 |
| D5 | 1 | GPIO18 | Pin 12 |
| D6 | 5 | GPIO24 | Pin 18 |
| D7 | 4 | GPIO23 | Pin 16 |

### 2.2 固定接线

- `VSS -> GND`
- `VDD -> 5V`
- `RW -> GND`
- `VO -> 电位器中间脚`
- `A/K -> 背光`

说明：

- `VO` 决定对比度，如果屏幕亮但没有字，优先调电位器
- `RW` 建议固定接地，使用只写模式

## 7. 最小显示示例

下面是一个最小可运行示例，可以在 LCD1602 上显示两行文字：

```python
import time
import RPi.GPIO as GPIO

LCD_RS = 22
LCD_E = 17
LCD_D4 = 25
LCD_D5 = 18
LCD_D6 = 24
LCD_D7 = 23

LCD_WIDTH = 16
LCD_CHR = True
LCD_CMD = False

LCD_LINE_1 = 0x80
LCD_LINE_2 = 0xC0

E_PULSE = 0.0005
E_DELAY = 0.0005


def lcd_toggle_enable():
    time.sleep(E_DELAY)
    GPIO.output(LCD_E, True)
    time.sleep(E_PULSE)
    GPIO.output(LCD_E, False)
    time.sleep(E_DELAY)


def lcd_send_byte(bits, mode):
    GPIO.output(LCD_RS, mode)

    GPIO.output(LCD_D4, bool(bits & 0x10))
    GPIO.output(LCD_D5, bool(bits & 0x20))
    GPIO.output(LCD_D6, bool(bits & 0x40))
    GPIO.output(LCD_D7, bool(bits & 0x80))
    lcd_toggle_enable()

    GPIO.output(LCD_D4, bool(bits & 0x01))
    GPIO.output(LCD_D5, bool(bits & 0x02))
    GPIO.output(LCD_D6, bool(bits & 0x04))
    GPIO.output(LCD_D7, bool(bits & 0x08))
    lcd_toggle_enable()


def lcd_init():
    GPIO.setwarnings(False)
    GPIO.setmode(GPIO.BCM)
    for pin in (LCD_E, LCD_RS, LCD_D4, LCD_D5, LCD_D6, LCD_D7):
        GPIO.setup(pin, GPIO.OUT)
        GPIO.output(pin, False)

    lcd_send_byte(0x33, LCD_CMD)
    lcd_send_byte(0x32, LCD_CMD)
    lcd_send_byte(0x28, LCD_CMD)
    lcd_send_byte(0x0C, LCD_CMD)
    lcd_send_byte(0x06, LCD_CMD)
    lcd_send_byte(0x01, LCD_CMD)
    time.sleep(E_DELAY)


def lcd_write_line(message, line_address):
    lcd_send_byte(line_address, LCD_CMD)
    message = message.ljust(LCD_WIDTH)[:LCD_WIDTH]
    for char in message:
        lcd_send_byte(ord(char), LCD_CHR)


lcd_init()
try:
    lcd_write_line("LCD1602 Ready", LCD_LINE_1)
    lcd_write_line("GPIO Test OK", LCD_LINE_2)
    time.sleep(8)
finally:
    lcd_send_byte(0x01, LCD_CMD)
    GPIO.cleanup()
```

## 10. 常见问题排查

如果 LCD 背光亮但没有文字，优先检查：

- `VO` 电位器是否调到合适位置
- `RW` 是否接地
- `RS/E/D4/D5/D6/D7` 是否接对

如果完全没有反应，优先检查：

- `VSS -> GND`
- `VDD -> 5V`
- 背光 `A/K` 是否连接正确
- 面包板供电是否稳定

如果显示乱码，优先检查：

- 数据线顺序是否正确
- 初始化时序是否被改动
- 是否多程序同时占用了这些 GPIO
