## 2. 当前硬件信息

当前舵机控制使用：

- GPIO 模式：`BCM`
- 舵机控制引脚：`GPIO27`
- PWM 频率：`50Hz`

Python 初始化方式：

```python
import RPi.GPIO as GPIO

SERVO_PIN = 27
PWM_FREQUENCY = 50

GPIO.setwarnings(False)
GPIO.setmode(GPIO.BCM)
GPIO.setup(SERVO_PIN, GPIO.OUT)

pwm = GPIO.PWM(SERVO_PIN, PWM_FREQUENCY)
pwm.start(0)
```

## 3. 角度与占空比换算

当前使用的换算公式如下：

```python
def angle_to_duty_cycle(angle):
    angle = max(0, min(180, angle))
    return 2.5 + (angle / 180.0) * 10.0
```

说明：

- 角度范围限制在 `0` 到 `180`
- `0°` 对应较低占空比
- `180°` 对应较高占空比
- 实际舵机机械极限可能小于理论值，必要时应适当收缩范围

## 5. 最小角度控制代码

下面是最小可运行代码，可以把舵机转到 `180°`：

```python
import time
import RPi.GPIO as GPIO

SERVO_PIN = 27
PWM_FREQUENCY = 50


def angle_to_duty_cycle(angle):
    angle = max(0, min(180, angle))
    return 2.5 + (angle / 180.0) * 10.0


GPIO.setwarnings(False)
GPIO.setmode(GPIO.BCM)
GPIO.setup(SERVO_PIN, GPIO.OUT)

pwm = GPIO.PWM(SERVO_PIN, PWM_FREQUENCY)
pwm.start(0)

try:
    pwm.ChangeDutyCycle(angle_to_duty_cycle(180))
    time.sleep(1.2)
    pwm.ChangeDutyCycle(0)
    time.sleep(0.2)
finally:
    pwm.stop()
    GPIO.cleanup()
```

## 9. 注意事项

### 9.1 供电

舵机动作时瞬时电流可能较大，若树莓派供电不足，可能出现：

- 抖动
- 无法到位
- 树莓派欠压

建议：

- 舵机单独供电
- 与树莓派共地

### 9.2 机械限位

虽然代码中可写 `0` 到 `180`，但实际舵机不一定能安全达到全角度。

如果出现以下情况，应减小角度范围：

- 明显卡住
- 有异响
- 抖动严重
- 发热明显

### 9.3 PWM 清理提示

在部分运行结束后，`RPi.GPIO` 可能会输出类似 `PWM.__del__` 的清理警告。

这通常发生在程序退出阶段，常见表现为：

- 动作已经执行成功
- 退出时出现额外警告

如果舵机动作已经完成，这类警告通常不影响本次控制结果。
