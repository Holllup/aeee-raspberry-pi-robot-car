# 🎥 一、摄像头初始化

代码：

```cpp
VideoCapture cap;
cap.open(0);
```

👉 含义：

* `VideoCapture` = OpenCV 摄像头接口
* `0` = 默认摄像头（一般就是 `/dev/video0`）

👉 本质：

```text
树莓派 → 通过 V4L2 → 读取 USB 摄像头视频流
```

---

# ⚙️ 二、分辨率设置

```cpp
cap.set(CV_CAP_PROP_FRAME_WIDTH, FRAME_W);
cap.set(CV_CAP_PROP_FRAME_HEIGHT, FRAME_H);
```

对应：

```cpp
const int FRAME_W = 320;
const int FRAME_H = 240;
```

👉 摄像头输出：

```text
320 × 240（低分辨率，提高实时性）
```

---

# 🔄 三、每一帧是怎么来的

主循环：

```cpp
Mat frame;
cap >> frame;
```

👉 这一步做了：

```text
摄像头 → 采一帧 → 存到 frame（Mat图像）
```

如果失败：

```cpp
if(frame.empty())
```

👉 就说明：

* 摄像头没数据
* 或断开

# 🧠 四、图像处理流程（核心）

调用：

```cpp
findLineCenter(frame, cx, cy);
```

---

## 🔍 1. 只取图像底部一条区域（提速关键）

```cpp
ROI_TOP_RATIO = 0.72
ROI_BOTTOM_RATIO = 0.95
```

👉 实际效果：

```text
只看画面底部约 23% 区域
```

原因：

* 黑线一定在车前下方
* 减少计算量

---

## 🎨 2. 转灰度 + 模糊

```cpp
cvtColor → 灰度
GaussianBlur → 去噪
```

---

## ⚫ 3. 提取黑线

```cpp
threshold(..., THRESH_BINARY_INV)
```

👉 意思：

```text
黑色 → 白色（255）
背景 → 黑色（0）
```

---

## 🧹 4. 形态学处理

```cpp
MORPH_CLOSE（填洞）
MORPH_OPEN（去噪）
```

👉 作用：

* 去掉杂点
* 保留线条

---

## 🔲 5. 找轮廓

```cpp
findContours()
```

👉 找所有白色区域

---

## 🎯 6. 选最大区域（认为是黑线）

```cpp
找面积最大的轮廓
```

过滤条件：

```cpp
area >= MIN_AREA_RATIO
```

---

## 📍 7. 求中心点

```cpp
Moments → 计算质心
```

得到：

```text
cx = 黑线中心 x
cy = 黑线中心 y
```

---

# 🎮 五、摄像头最终输出给控制系统

```text
摄像头 → 图像 → 黑线 → 中心点(cx)
```

然后：

```cpp
error = cx - 图像中心
```

👉 用来控制转向
